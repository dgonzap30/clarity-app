# Spending Dashboard

A comprehensive React + Bun dashboard for analyzing AMEX spending transactions with beautiful charts, budget tracking, and category breakdowns.

## Features

- **Summary Cards** - Total spending, transaction count, averages
- **Category Donut Chart** - Visual breakdown of spending by category
- **Monthly Trend Chart** - Bar chart showing spending over time
- **Budget Progress Bars** - Track progress against monthly budgets
- **Transaction List** - Searchable and filterable transaction table
- **CSV Upload** - Drag and drop to upload new transaction data
- **PDF Export** - Generate printable spending reports
- **Dark Mode** - Beautiful dark theme by default with light mode toggle

## Getting Started

1. Install dependencies:
   \`\`\`bash
   bun install
   \`\`\`

2. Start the development server:
   \`\`\`bash
   bun run dev
   \`\`\`

3. Open http://localhost:5173 in your browser

## Tech Stack

- Bun + React 18 + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Recharts for charts
- jsPDF + html2canvas for PDF export
